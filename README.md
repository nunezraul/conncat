# conncat
5630124154
ConnCat is an Android application designed to connect UC Merced students with events at school as well as in the local community.

### How it works
* Simple and easy view of latest events
* Relevant filtering
* Events aggregation
* Add your own event

### Features
* View by on-campus/off-campus
* Map view
* View by categories
* Search
